# MireDigital

This repository is a practice project for learning and experimenting with Git, GitHub, and version control workflows. It includes various tasks and exercises to enhance my understanding of:

- Git basics (commits, branches, merges, etc.)
- GitHub features (pull requests, issues, actions, etc.)
- Version control best practices
- Collaboration workflows

## Features

Hands-on practice with Git commands
Experimentation with GitHub features
Learning collaboration workflows
